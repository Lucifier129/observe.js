observe.js
==========

观察对象的属性，当它变化时，调用指定函数。

<code>var test = observe({ background: '#000'});</code>

<code>test.on('background', function(key, value) {
  document.body.style[key] = value;
});</code>

<code>setTimeout(function() {
  test.background = '#f00';
}, 1000);</code>

document.body的背景颜色，将在一秒后变成#f00.
